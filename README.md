# Snip — Smart URL Shortener

A URL shortener with click analytics built with PHP + MySQL.

## Features
- User registration and login
- URL shortening with unique short codes
- Click tracking (browser, device, IP)
- Analytics dashboard
- Forgot password with OTP via email

## Tech Stack
- PHP 8+ (OOP)
- MySQL
- HTML + CSS + JavaScript
- Brevo API (email)

## Setup
1. Clone the repo
2. Copy `config/database.php` replace Placeholder with actual value
3. Fill in your DB credentials and Brevo API key
4. Import the SQL tables from `database.sql`
5. Upload to any PHP hosting
